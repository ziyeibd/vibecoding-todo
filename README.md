\# 我的待办清单



这是我学习 vibecoding 过程中做的第一个小工具。



\## 功能



\- 添加待办任务

\- 标记完成/取消完成

\- 删除单个任务

\- 清空全部任务

\- 刷新页面后保留任务



\## 使用方法



1\. 打开 `index.html`

2\. 在输入框里输入一个任务

3\. 点击“添加任务”

4\. 可以点击“完成/取消”切换任务状态

5\. 可以点击“删除”移除任务

6\. 可以点击“清空全部任务”清空列表



\## 我学到的内容



\- HTML 负责页面结构

\- CSS 负责页面样式

\- JavaScript 负责页面行为

\- 数组可以保存多条数据

\- 对象可以保存一条任务的多个信息

\- localStorage 可以把数据保存在浏览器里

\- render 函数可以把数据渲染到页面上
